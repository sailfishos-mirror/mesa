# Mali counter database

All the files under this directory have been directly copied from
(libGPUCounter)[https://github.com/ARM-software/libGPUCounters/tree/main/specification/database]
and are subject to the original Arm copyright that applies to this
project licensed under MIT.

If you need to update the counters definition, copy the new files
in, and make sure mali_gen_perf.py is still able to parse them.
